Meet HTML Tags 🏷️

HTML tags are the special instructions we give to the browser. They tell the browser what to show and how to show it.

Think of tags like labels on boxes. Each label tells the browser exactly what is inside.

Example of a Tag

<h1>Hello</h1>

This tells the browser to show a big heading.

The Tag Sandwich 🥪

Most HTML tags come in pairs. One opens the instruction, and the other closes it. Just like bread on both sides of a sandwich.

<h1> My First Website </h1>

Opening Tag

Starts the instruction

Content

What people see

Closing Tag

Ends the instruction

Common HTML Tags You Must Know 🚀

<h1>

Heading

Used for big titles like page headings.

<p>

Paragraph

Used for writing normal text and stories.

<img>

Image

Used for putting pictures on your webpage.

<a>

Anchor Link

Used for links that move users to another page.

<button>

Button

Used for clickable buttons on websites.

<div>

Container

Used for grouping sections together.

Quick Practice ✍️

Which tag should you use for a big title like: "Welcome to My Website"?

<h1>
<p>
<img>

Correct answer: <h1>
